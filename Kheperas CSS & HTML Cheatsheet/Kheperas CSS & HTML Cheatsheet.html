This table is designed to help you quickly identify CSS properties to be used in your projects.

CSS CheatSheet

Code

Description
font-family: Allows you to set fontface
font-weight: Sets the thickness of the font
font-size: Sets the size of the font
font-style: Allows you to italicise the font
color: Sets the color of the text
background-color: Sets the color of the background
of the text
background-image: Allows you to set a background image, must follow with url and ();
text-align: Aligns the text
Left, Center or Right
text-decoration: Allows you to add line-through or underline to text
margin: Sets margin for all 4 sides starting with the top, e.g.
margin: 8px 4px 8px 4px;
height: Defines the height of an element
width: Defines the width of an element
padding: Defines padding for all 4 sides starting with the top, e.g.
padding: 10px 10px 10px 10px;
border: Defines a border around an element
set values for width, style & color, e.g.
border: 2px solid black;

HTML

This table is designed to give you a refresher on the different types of HTML tags; you know, just in case!

HTML Reference

Tags

Description
<html> </html> Defines page type, all elements are contained in these tags.
<head> </head> Contains important information and tags such as:
meta, title, links & may contain author information with contact info.
<title> </title> Defines the title of the html document and is to be placed inside the head tags. The title is displayed in the browsers tab.
<link> Used to link a CSS file to an html document, it is a self closing tag.
<body> </body> Contains the bulk of the content.
<header> </header> Used to display introductory content.
<main> </main> Specifies main content of the html document and is to be used once.
<div> </div> Creates a container for content; representing a division or section of content.
<h1> </h1>
...
<h6> </h6>		 Used for the headings of your content, ranging from h1 to h6, with h1 being the largest header, and h6 being the smallest.
<p> </p> This tag is used to contain all the pieces of text that serve as the actual readable content in the document.
<br> Pushes the text onto the next line, much like the enter key in text editors. It is a self-closing tag.
<hr> Creates a line used to separate sections, you can see them used on this page, before and after each table. It is a self-closing tag.
<img src=""> Used to place images in the document from a local file on your computer or by using an image hosting company. The link or source is placed inbetween the quotation marks.
<a href=""> </a> Used to add links to the page. Note that you can also wrap other elements with links, if you wanted a picture to be a link for example.
